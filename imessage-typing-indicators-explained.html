How iMessage Typing Indicators Work and How Long They Last

With typing indicators in iMessage, you can see when other Apple device users are writing a message back to you. This is helpful as it lets us know if someone saw a message and whether they're quickly typing a response.

You may have also heard people call them texting bubbles, message bubbles, or typing bubbles. While there are many names to call typing indicators, there are also many questions. Read ahead if you want to know about typing indicators.

How Do Typing Indicators Work?

Typing indicators work over an active Wi-Fi or cellular connection; once you start typing, it lets the recipient's iPhone know that a message is being written.

When this occurs, the Messages app triggers the bubble notification. This notification will also activate if you're trying to send a picture, so it isn't just for typing.

Typing indicators only work if you and the recipient both have an Apple device and are using iMessage to text each other. However, if you're not connected to the internet while you're typing, the typing indicator wouldn't show up for the recipient.

How Long Do Typing Indicators Last?

According to several tests, typing indicators last about 60 seconds. Once you start typing, the 60-second clock begins. After the 60-second clock is up, the typing indicators will disappear. Even if you're still typing, the recipient won't see it after 60 seconds.

The recipient will still see the typing indicator if you type a message and leave it in the box without sending it. However, whether you send the message or not, the indicator will disappear after 60 seconds.

If you go back to an unsent message later, the typing indicators won't show up either. Only once you send a message will the typing indicator return.

Can You Turn Off Typing Indicators?

Currently, you can't turn off typing indicators in iMessage, similar to how you'd disable read receipts. Hopefully, in the future, Apple will give users the option to do so.

However, If you don't want to send a typing indicator, there is a workaround. Since it requires an internet connection, you can simply put your iPhone into airplane mode before typing. Once you do that, it won't send a typing indicator. However, you won't receive any either.

Will Apple Let Us Turn Off Typing Indicators?

Typing indicators or texting bubbles, whatever you want to call them, show other people when you're actively typing a message. This is helpful as it prevents typing over one another and creates a more streamlined texting thread.

However, some people don't want others to know when they're typing a message. Because of this, Apple may allow us to turn off typing indicators in the future. Until then, you can always use Airplane Mode to hide when you're typing.
